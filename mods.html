Tropik Modding Guide

Available APIs

  • TropikAPI.setPlayerSpeed(player, speed) - Modify player movement
  • TropikAPI.spawnParticles(position, type) - Create visual effects
  • TropikAPI.playSound(filename) - Play audio files
  • TropikAPI.setGravity(player, gravity) - Adjust gravity
  • TropikAPI.showMessage(player, text) - Display messages
  • TropikAPI.giveItem(player, item) - Give items to players

Events

  • player_spawn - When a player joins the game
  • island_enter - When a player enters an island
  • axolotl_interact - When interacting with axolotls
  • npc_talk - When talking to NPCs
  • zipline_use - When using ziplines
